📖 About This Project

KORA (which means "Work" in Kinyarwanda) was built as a web engineering assignment at the African Leadership University in Kigali, Rwanda. The goal was to create a real-world, data-driven application that solves a genuine problem for Rwandan professionals.

Growing up in Rwanda, I noticed how difficult it was to find quality job listings in one place — especially for Africa-specific roles. Most global job boards are built for Western markets. KORA bridges that gap by aggregating listings from multiple sources and presenting them through an Africa-first lens.

The project integrates three live job APIs (Adzuna, JSearch, and Remotive), visualises skills and salary trends with Chart.js, and is deployed to production on two Ubuntu web servers behind an HAProxy load balancer.
